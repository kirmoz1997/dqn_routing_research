# Tools

## tsv_to_jsonl.py

Скрипт конвертирует `data/tasks_draft.tsv` в `data/tasks.jsonl` (JSONL: один JSON-объект на строку).

### Формат входного TSV

Файл должен быть в UTF-8 и содержать заголовок:

```
id	label	difficulty	eval_hint	text	notes
```

Допускаются пустые строки — они будут пропущены.

### Валидация

Скрипт останавливается с ошибкой, если:

- заголовок не совпадает с ожидаемым;
- число колонок в строке не равно 6;
- `id` пустой или повторяется;
- `label` не является целым числом или не входит в диапазон 0..8.

При ошибке выводится сообщение с номером строки, после чего выполнение завершается.

### Выходной формат

Результат сохраняется в `data/tasks.jsonl`.

- кодировка UTF-8;
- `ensure_ascii=False` (без ASCII-экранирования);
- одна запись на строку.

### Запуск

Из корня проекта:

```
python tools/tsv_to_jsonl.py
```
